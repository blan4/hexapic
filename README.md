#Hexapic

Written by @blan4

Download last 6 pictures from instagram, generate collage and set it as desktop wallpaper.

##Example usage

###Search for pictures with tag 'cat'

```hexapic -t=cat```

###Search for user's pictures with name 'instagram'

```hexapic -u=instagram```

###Check updates of hexapic program

```hexapic -c```
